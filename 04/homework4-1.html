Question 1. Give the SELECT command to answer the question Who will go to the party held in Peking University?

SELECT guest.Guest_Name
FROM guest, party, guest_party
WHERE party.Party_Num=guest_party.Party_Num
AND guest_party.Guest_Num=guest.Guest_Num
AND party.Place='Peking University';

Question 2. Figure out a way to obtain the column labels returned by mysql_fetch_array.

$result = mysql_query($your_query);
$row = mysql_fetch_array($result);
while($pair = each($row)) {
 $key = $pair["key"];
 if (is_string($key)) print("$key");
}

Question 3. Tell the difference between a CGI program and a PHP program executed by a Web Server.

PHP script is embedded in HTML documents, and is processed by an Apache module upon requested, of which the result replaces the script and forms a complete HTML document.

CGI program is invoked by Apache upon requested, and they use a predefined method to communicate. Apache reads output from the CGI program and regards it as a complete document. CGI program can be written in any language and executed in any method.

Lei Yang, Fall 2016
